# Utilities Library

This directory contains the implementation of the broad utility functions in the **rice2025** Python package.

--- 

## Overview

The utilities module provides implementations of common utilities used in machine learning algorithms.

## Subpackage Structure
```
utilities
├── metrics.py              # Implements euclidean and manhattan distances
├── postprocess.py          # Implements majority and average labeling
├── preprocess.py           # Implements scaling, normalizing, and train/test splitting
└── README.md               # This file
```
