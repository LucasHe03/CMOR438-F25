# Supervised Learning Library

This directory contains the implementation of the supervised learning algorithms in the **rice2025** Python package.

--- 

## Overview

The supervised learning module provides implementations of common supervised learning algorithms and supporting utilities.

## Included Algorithms

- Perceptron
- Multi-layer Perceptron
- Logistic Regression
- Linear Regression
- K-Nearest Neighbors
- Decision Trees & Regression Trees
- Random Forests
- Ensemble Methods
